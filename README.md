# Intelligent Complaint Analysis for Financial Services

This project implements a Retrieval-Augmented Generation (RAG) chatbot to analyze customer complaints from the Consumer Financial Protection Bureau (CFPB) dataset.

## Business Objective
To enable product, support, and compliance teams to quickly identify emerging complaint trends across financial products such as credit cards, personal loans, savings accounts, and money transfers.

## Key Features
- Semantic search over customer complaints using vector databases
- Evidence-backed answers powered by large language models
- Multi-product complaint analysis
- Interactive chat interface for non-technical users

## Project Structure
See the repository structure for data preprocessing, vector indexing, RAG pipeline logic, and UI implementation.

## Tech Stack
- Python
- Sentence Transformers
- ChromaDB / FAISS
- LangChain
- Gradio / Streamlit
